## Browse publications

On this page we had show you how to add bookmarks and publications to BibLicious. In this tutorial you will learn how to search for bookmarks / publications.

__Requirements:__ To use this tutorial you must unlock the "advanced features" of BibLicious - How to activate the advances features .

![](1.png)

__Instructions:__

  1. Move the mouse to __"myBibLicious"__. A menu appears. Then click on __"browse publications"__ ( Note: This menu item is only visible if you had activated the advanced features|Unlock advanced mode] have ) .
  2. __"search options"__ allows you to choose different tags and options, for which you want to display all entries. In order to select more than just a list item, hold down the CTRL key pressed during the mouse click.
  3. You can use the __"and/or"__ buttons to set up the search query in different ways.
  4. Under __"search results"__ you will find all results that match the selection of (2 and 3).
  5. The textbox __"filter"__ allows you to enter a part of the publication/bookmark title, for a text match filtering of the results.
